== Installation ==
Uploading Through WordPress
To upload through your Dashboard, select “Plugin” . Next, select the “Add New” button at the top of the page and then select the “Upload Plugin” button. Choose BS Maps - Google Map and Leaflet Map WordPress Plugin for Elementor and WPBackery "bsmaps.zip" file and select “Install Now”. Once the plugin is fully uploaded and installed, click “Activate” to activate the WordPress plugin.

When uploading your plugin with the installer, please ensure you are uploading the plugin .zip file, not the entire package you downloaded. In this case, you will be uploading "bsmaps.zip".
FTP Upload
To manually upload your new WordPress plugin, login with your credentials to your website and locate the wp-content folder in your WordPress install files. Upload the un-zipped BS Maps - Google Map and Leaflet Map WordPress Plugin for Elementor and WPBackery folder into the: wp-content/plugins folder.
Once uploaded, activate the plugin by heading to the “Installed Plugins” menu in the WordPress Dashboard.

Installing Plugins Required
Once you have installed and activated our plugin you should install required plugins for proper work of plugin . The Plugin is addon for Elementor and WPBackery Page Builder . Therefore, Elementor and WpBackery Page Builder plugins should be installed and activated.== Screenshots ==
